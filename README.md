# 🎮 Rock Paper Scissors Game

A modern, responsive Rock Paper Scissors game with score tracking and beautiful UI design.

## ✨ Features

- 🎯 **Fully Functional Game**: Classic Rock Paper Scissors gameplay
- 📊 **Score Tracking**: Keeps track of user wins, computer wins, and ties
- 🔄 **Reset Function**: Reset the game and scores at any time
- 📱 **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- 🎨 **Modern UI**: Beautiful gradients, animations, and hover effects
- ⌨️ **Keyboard Support**: Press Enter to play after typing your choice
- 🎭 **Visual Feedback**: Different colors for wins, losses, and ties

## 🚀 How to Play

1. Open `index.html` in your web browser
2. Type your choice: `rock`, `paper`, or `scissors`
3. Click "click here" button or press Enter
4. See the result and updated scores
5. Click "Reset Game" to start over

## 🎯 Game Rules

- **Rock** beats **Scissors**
- **Scissors** beats **Paper**
- **Paper** beats **Rock**
- Same choice = **Tie**
